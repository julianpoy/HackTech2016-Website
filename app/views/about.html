Install

The Following steps will get you up and running with Flyxer!

  1. Download a build from the Latest Releases on the Flyxer Github Repository
  2. Extract the contents of the .zip into a folder of your liking. If you have a Mac, place the Flyxer.app into your applications folder.
  3. Run the file named "app", from the directory you extracted to. If you have a mac, run Flyxer from your applications folder.
  4. Enjoy the software, and play your music loud!
P.S if you dont have your own songs, don't worry, a couple of demo tracks are included under the demo tracks folder.
